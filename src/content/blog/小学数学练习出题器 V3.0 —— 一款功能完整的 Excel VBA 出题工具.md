# 小学数学练习出题器 V3.0 —— 一款功能完整的 Excel VBA 出题工具

## 一、工具简介

这是一款基于 Excel VBA 开发的**小学数学练习出题器**，专门为小学生设计，可以帮助老师或家长快速生成各类数学练习题。

**核心特点**：
- ✅ 完全免费，只需 Excel 即可运行
- ✅ 参数高度可调，满足不同年级需求
- ✅ 支持两数运算 + 多步混合运算
- ✅ 排版灵活，可直接打印使用

---

## 二、功能模块介绍

### 2.1 通用设置

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 题目数量 | 一次生成多少道题 | 20~50 |
| 每行题数 | 一行显示几道题 | 2~3 |
| 列宽 | 每列的宽度（磅） | 35 |
| 题目行高 | 题目所在行的高度 | 30 |
| 空白行高 | 竖式区空白行的高度 | 40 |
| 显示题号 | 每题前是否显示序号 | ✅ |
| 留竖式空白区域 | 是否留出手写竖式的空白行 | ✅ |
| 显示边框 | 是否显示表格虚线边框 | ✅ |

### 2.2 加法设置

| 参数 | 说明 |
|------|------|
| 启用加法 | 是否生成加法题 |
| 加数1范围 | 第一个加数的取值范围 |
| 加数2范围 | 第二个加数的取值范围 |
| 和的最大值 | 计算结果的上限 |
| 允许进位 | 个位相加是否允许 ≥10 |
| 随机交换加数 | 随机显示 a+b 或 b+a |

### 2.3 减法设置

| 参数 | 说明 |
|------|------|
| 启用减法 | 是否生成减法题 |
| 被减数范围 | 被减数的取值范围 |
| 减数范围 | 减数的取值范围 |
| 差的最小值 | 结果必须 ≥ 此值 |
| 允许退位 | 个位相减是否允许借位 |
| 允许负数结果 | 是否允许被减数小于减数 |

### 2.4 乘法设置

| 参数 | 说明 |
|------|------|
| 启用乘法 | 是否生成乘法题 |
| 乘数1范围 | 第一个乘数的取值范围 |
| 乘数2范围 | 第二个乘数的取值范围 |
| 积的最大值 | 乘积的上限 |
| 随机交换乘数 | 随机显示 a×b 或 b×a |

### 2.5 除法设置

| 参数 | 说明 |
|------|------|
| 启用除法 | 是否生成除法题 |
| 被除数范围 | 被除数的取值范围 |
| 除数范围 | 除数的取值范围（自动排除0） |
| 要求整除 | 不允许有余数 |

> **有余数时的记法**：`17 ÷ 5 = 3 ...... 2`

### 2.6 混合运算设置（独立）

| 参数 | 说明 |
|------|------|
| 启用混合运算 | 启用后忽略上方运算选择 |
| 数字范围 | 参与运算的数字范围 |
| 允许的运算 | 加法/减法/乘法/除法（可多选） |
| 运算步数 | 几个数字参与（2~6） |
| 最终结果范围 | 结果必须在此范围内 |
| 结果必须为整数 | 除法时要求整除 |
| 中间结果非负 | 每一步结果都 ≥0 |
| 运算顺序 | 从左到右 / 先乘除后加减 |

## 五、使用步骤

### 第一步：设置参数
- 根据学生年级和练习需求，调整各区域参数
- 混合运算需要单独勾选启用

### 第二步：生成题目
- 点击 **生成题目** 按钮
- 题目自动生成到“练习题”工作表中

### 第三步：打印或使用
- 可直接在 Excel 中练习
- 点击 **打印预览** 调整后打印

### 第四步：清空重置
- 点击 **清空练习区** 可清除当前题目，重新生成

## 六、分年级推荐配置

| 年级 | 推荐设置 |
|------|----------|
| **一年级** | 加法、减法（不进位/不退位），数字范围 0~20，和≤20，不留竖式区 |
| **二年级** | 加法（进位）、减法（退位）、表内乘法，数字范围 0~100 |
| **三年级** | 加减乘除混合（从左到右），数字范围 0~100，可留竖式区 |
| **四年级以上** | 多步混合运算（先乘除后加减），步数 3~4，含除法有余数 |

## 八、总结

这款小学数学练习出题器有以下优势：

1. **完全免费**：只要有 Excel 就能运行
