# 博客分类体系文档

![网站博文分类.jpg](https://image.zyglz.com/file/zyglz/2026/1773459965517.jpg)

## 📌 分类概述

本博客采用**五维分类体系**，涵盖实用工具、资源分享、技术笔记、网站运营、生活随笔五大领域。每个主分类下设四个子分类，共计20个细分栏目，既保证内容覆盖全面，又确保分类边界清晰。

---

## 一、实用工具

**定位**：各类软件、在线工具、网站服务的推荐与评测


| 子分类  | 内容说明                         | 文章示例                 |
| ---- | ---------------------------- | -------------------- |
| 电脑软件 | Windows/Mac/Linux各类软件推荐、使用技巧 | 效率神器推荐、破解软件分享、软件使用教程 |
| 在线工具 | 无需安装的Web工具、在线服务              | 在线PS、在线转换、在线压缩工具推荐   |
| 手机应用 | iOS/Android APP推荐与评测         | 实用APP合集、安卓神器、iOS必备应用 |
| 网站推荐 | 优质网站、资源站、导航站                 | 资源搜索网站、免费素材站、小众导航推荐  |


**适合文章**：原“软件应用”“网站推荐”类内容

---

## 二、资源分享

**定位**：电子书、模板、素材、学习资料等可直接下载的资源


| 子分类  | 内容说明              | 文章示例                     |
| ---- | ----------------- | ------------------------ |
| 电子书籍 | PDF、EPUB等格式的电子书资源 | 编程电子书合集、经典书籍分享、 Kindle资源 |
| 设计素材 | 图片、图标、字体、模板等      | PPT模板、UI套件、免费字体、免抠图素材    |
| 学习资料 | 教程、笔记、试题、文档等      | 考证资料、学习笔记、课程资源、技术文档      |
| 资源合集 | 各类资源的汇总整理         | 百大资源网站、年度精选合集、分类导航       |


**适合文章**：原“资源分享”“资源集合”类内容

| 子分类  | 内容说明                      | 文章示例                          |

| 折腾搭建 | 自己搭建的各种服务       | 自建导航站、短链接服务、网盘搭建、图床建设       |

## 🏷️ 标签系统建议

分类是**大方向**，标签是**细节点**。每篇文章建议添加3-5个标签：


| 维度   | 标签示例                                                 |
| ---- | ---------------------------------------------------- |
| 技术标签 | #Proxmark3 #RFID #NFC #硬件安全 #破解教程 #编程 #Python #Linux |
| 工具标签 | #软件推荐 #在线工具 #神器 #效率工具 #APP推荐                         |
| 资源标签 | #电子书 #模板 #素材 #免费资源 #设计素材                             |
| 网站标签 | #建站教程 #SEO #服务器 #WordPress #自建项目                     |
| 生活标签 | #随笔 #感悟 #读书笔记 #旅行 #日常记录                              |

## ✅ 分类体系优势


| 特点        | 说明                                          |
| --------- | ------------------------------------------- |
| **全面性**   | 5大主分类 × 4子分类 = 20个细分栏目，覆盖工具、资源、技术、运营、生活五大领域 |
| **清晰度**   | 每个分类边界明确，不会出现一篇文章不知道该放哪的情况                  |
| **专业性**   | 技术笔记单独成类，突出Proxmark3等硬核技术内容                 |
| **扩展性**   | 每个主分类下设4个子分类，未来可继续扩充                        |
| **统一性**   | 所有分类均为四字，界面整齐美观                             |
| **SEO友好** | 分类名称简洁明了，便于搜索引擎抓取                           |

## 🔄 分类维护原则

1. **一篇文章一个主分类**：避免跨分类，保持内容聚焦
2. **标签作为补充**：用标签关联相关内容
3. **定期评估**：每半年评估一次分类是否合理
4. **避免过度细分**：子分类保持在4个以内，避免选择疲劳
5. **保持一致性**：新文章严格按照此分类发布
